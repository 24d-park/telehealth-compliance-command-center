## Verified requirements (12 state-sourced + federal-framework inheritance)

Full IL licensure required for telehealth, no separate telehealth registry (225 ILCS 150/10);
IL is notably PERMISSIVE on modality — "Interactive telecommunications system" expressly includes
audio-only telephone, and telehealth services include store-and-forward (225 ILCS 150/5); no
in-person-first mandate (225 ILCS 150/15); Ryan Haight Act adopted for internet CS prescribing
(68 Ill. Adm. Code 1330.60); PMP registration + documented access for initial Schedule II opioids
(720 ILCS 570/314.5(c-5)); nonresident pharmacy license incl. Internet pharmacies (225 ILCS 85/16a);
Sch II 30-day-supply / 90-day-validity limits (720 ILCS 570/312); PIC (225 ILCS 85/15); USP-NF 2024
(USP 47-NF 42) compounding adopted by rule (68 Ill. Adm. Code 1330.640).
